# 🏋️ Road to Glory

A personal gym tracking app for **Abdulhadi** and **Abdullah**.

## Files

| File | Description |
|------|-------------|
| `index.html` | Splash screen — Road to Glory landing page |
| `workout.html` | Full 6-day workout schedule with weight logging |

## How to use

1. Open `index.html` in your browser (or host on GitHub Pages)
2. Tap **ENTER THE GYM** to start logging workouts
3. App auto-detects the correct day based on the date:
   - Day 1 → Saturday (Heavy)
   - Day 2 → Sunday (Volume)
   - Day 3 → Monday (Pump)
   - Day 4 → Tuesday (Heavy)
   - Day 5 → Wednesday (Volume)
   - Day 6 → Thursday (Athletic)
   - Day 7 → Friday (Rest)
4. Weights are saved locally in the browser

## Deploy on GitHub Pages

1. Upload both files to a GitHub repo
2. Go to **Settings → Pages**
3. Set source to `main` branch, root folder
4. Your app will be live at `https://yourusername.github.io/repo-name`
